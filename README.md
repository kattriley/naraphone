# Nara Phone

A lightweight multi-tab Android WebView browser with full translation support (English / Nederlands / Polski).

## Features

- **Multi‑tab browsing** — open, switch, and close tabs with × buttons
- **Bottom navigation** — back, forward, reload, new tab, settings
- **Quick‑link shortcuts** — Google, Pinterest, Roblox, Discord, Spotify
- **Dark mode** — toggle with persistent preference
- **Language support** — English, Nederlands, Polski (switch in settings)
- **Address bar** — type URLs or search terms, Go button or keyboard action
- **Minimal & fast** — no extra permissions, lightweight APK (~3 MB)

## Requirements

- Android 8.0 (API 26) or higher
- Internet connection

## Build

```bash
# Clone the repo
git clone https://github.com/kattriley/naraphone.git
cd naraphone

# Build the debug APK
./gradlew assembleDebug

# APK location: app/build/outputs/apk/debug/app-debug.apk
```

**Note:** Requires JDK 17+ and Android SDK platform 35.
